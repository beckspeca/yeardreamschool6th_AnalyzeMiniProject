# 서울 지하철 승하차 인원과 따릉이 이용률 분석

## 1. 프로젝트 개요

본 프로젝트는 **서울 지하철 승하차 인원 데이터**와 **서울시 공공자전거 따릉이 이용 데이터**를 결합하여, 지하철역 주변의 따릉이 이용 패턴을 분석하는 데이터 분석 프로젝트입니다.

기존 지하철 승하차 인원 분석 프로젝트에 따릉이 이용 데이터를 추가하여, 단순히 지하철 혼잡도가 높은 역을 찾는 것에서 더 나아가 **지하철 하차 인원 대비 따릉이 이용률**과 **출퇴근 시간대별 따릉이 이용 특성**을 확인하는 방향으로 확장했습니다.

분석의 핵심 질문은 다음과 같습니다.

- 지하철 이용객이 많은 역일수록 따릉이 이용건수도 많은가?
- 출근 시간대와 퇴근 시간대에서 따릉이 이용 패턴은 다르게 나타나는가?
- 지하철 하차 인원 대비 따릉이 이용률이 높은 역은 어디인가?
- 따릉이가 지하철과 연계되는 “마지막 이동수단”으로 활용되는 경향이 있는가?

---

## 2. 사용 데이터

### 2.1 서울시 지하철 호선별 역별 승하차 인원 정보

- 파일명 예시: `Seoul_subway_data_20210705.csv`
- 주요 컬럼
  - `사용월`
  - `호선명`
  - `지하철역`
  - `04시-05시 승차인원`
  - `04시-05시 하차인원`
  - ...
  - `23시-24시 승차인원`
  - `23시-24시 하차인원`

이 데이터는 지하철역별, 시간대별 승차 인원과 하차 인원을 포함합니다. 분석에서는 2021년 6월 데이터를 기준으로 사용했습니다.

### 2.2 서울특별시 공공자전거 이용정보 시간대별 데이터

- 파일명 예시: `서울특별시 공공자전거 이용정보(시간대별)_21.06.csv`
- 주요 컬럼
  - `대여일자`
  - `대여시간`
  - `대여소명`
  - `이용건수`

이 데이터는 따릉이 대여소별, 시간대별 이용건수를 포함합니다. 분석에서는 대여소명에 포함된 지하철역 이름을 추출하여 지하철 데이터와 병합했습니다.

### 2.3 지하철역 위치 좌표 데이터

- 파일명 예시: `subway_location_data.csv`
- 주요 컬럼
  - `지하철역`
  - `x좌표`
  - `y좌표`

지도 시각화를 위한 보조 데이터입니다. Folium을 활용해 특정 호선의 혼잡도를 지도 위에 표시할 수 있습니다.

## 3. 분석 목표

본 프로젝트의 분석 목표는 다음과 같습니다.

1. 지하철 시간대별 승하차 인원 데이터를 전처리한다.
2. 따릉이 대여소명에서 지하철역명을 추출한다.
3. 지하철역명과 시간대를 기준으로 두 데이터를 병합한다.
4. 지하철 승차 인원, 하차 인원과 따릉이 이용건수 간의 관계를 분석한다.
5. 출근 시간대와 퇴근 시간대의 상관관계를 비교한다.
6. 지하철 하차 인원 대비 따릉이 이용률을 계산한다.
7. 출퇴근 시간대와 비출퇴근 시간대의 시간당 따릉이 이용률을 비교한다.

## 4. 분석 흐름

## 4.1 따릉이 데이터 전처리

따릉이 데이터에서 분석에 필요한 컬럼만 추출했습니다.

```python
ddr2 = ddr.loc[:, ["대여일자", "대여시간", "대여소명", "이용건수"]]
```

대여소명 끝에 붙어 있는 지하철역명을 정규표현식으로 추출했습니다.

```python
ddr2["지하철역"] = ddr2["대여소명"].str.extract(r"([가-힣0-9\.]+역)$")
```

예를 들어 다음과 같은 대여소명에서 지하철역명을 추출합니다.

| 대여소명 | 추출 결과 |
|---|---|
| 강남역 | 강남역 |
| 신림역 | 신림역 |
| 4.19민주묘지역 | 4.19민주묘지역 |

추출된 역명에서 `역` 글자를 제거하여 지하철 승하차 데이터와 병합할 수 있도록 정제했습니다.

```python
ddr2_cleaned = ddr2.dropna(subset=["지하철역"]).copy()
ddr2_cleaned["지하철"] = ddr2_cleaned["지하철역"].str.replace(r"역$", "", regex=True)
```

이후 날짜, 시간, 지하철역 기준으로 따릉이 이용건수를 합산했습니다.

```python
df_result = (
    ddr2_cleaned
    .groupby(["대여일자", "대여시간", "지하철"])["이용건수"]
    .sum()
    .reset_index()
)
```

마지막으로 2021년 6월 전체 기간을 시간대와 역 기준으로 다시 합산했습니다.

```python
df_hourly_total = (
    df_result
    .groupby(["대여시간", "지하철"])["이용건수"]
    .sum()
    .reset_index()
)
```

## 4.2 지하철 승하차 데이터 전처리

지하철 데이터는 시간대별 승하차 인원이 여러 컬럼으로 가로로 펼쳐져 있습니다. 분석을 위해 이를 세로 형태로 변환했습니다.

먼저 역명에서 괄호와 `역` 글자를 제거했습니다.

```python
df_subway["역명_정제"] = (
    df_subway["지하철역"]
    .str.replace(r"\(.*\)", "", regex=True)
    .str.replace(r"역$", "", regex=True)
)
```

시간대별 승차인원과 하차인원 컬럼만 추출했습니다.

```python
value_cols = [
    c for c in df_subway.columns
    if "승차인원" in c or "하차인원" in c
]
```

`melt()`를 사용해 시간대 컬럼을 행 방향으로 변환했습니다.

```python
df_subway_melted = pd.melt(
    df_subway,
    id_vars=["역명_정제"],
    value_vars=value_cols,
    var_name="시간대_유형",
    value_name="인원수",
)
```

`시간대_유형` 컬럼에서 시간과 승하차 유형을 분리했습니다.

```python
df_subway_melted["대여시간"] = (
    df_subway_melted["시간대_유형"]
    .str.extract(r"(\d+)시-")
    .astype(int)
)

_df_type = df_subway_melted["시간대_유형"].apply(
    lambda x: "승차인원" if "승차" in x else "하차인원"
)
df_subway_melted["유형"] = _df_type
```

이후 `pivot_table()`을 사용해 승차인원과 하차인원을 다시 별도 컬럼으로 분리했습니다.

```python
df_subway_pivot = (
    df_subway_melted
    .pivot_table(
        index=["대여시간", "역명_정제"],
        columns="유형",
        values="인원수",
        aggfunc="sum"
    )
    .reset_index()
)
```

## 4.3 따릉이 데이터와 지하철 데이터 병합

따릉이 데이터와 지하철 승하차 데이터는 `대여시간`과 `지하철역명`을 기준으로 병합했습니다.

```python
df_final = pd.merge(
    df_hourly_total,
    df_subway_pivot,
    left_on=["대여시간", "지하철"],
    right_on=["대여시간", "역명_정제"],
    how="inner",
)

# 최종 분석 컬럼 정리
df_final = df_final[
    ["대여시간", "지하철", "이용건수", "승차인원", "하차인원"]
].copy()
```

최종 데이터프레임 구조는 다음과 같습니다.

| 컬럼명 | 설명 |
|---|---|
| `대여시간` | 따릉이 대여 시간대 |
| `지하철` | 정제된 지하철역명 |
| `이용건수` | 해당 시간대, 해당 역 주변 따릉이 이용건수 |
| `승차인원` | 해당 시간대, 해당 역의 지하철 승차 인원 |
| `하차인원` | 해당 시간대, 해당 역의 지하철 하차 인원 |

corr_matrix = df_final[["이용건수", "승차인원", "하차인원"]].corr()

df_morning = df_final[df_final["대여시간"].isin([7, 8, 9])]

df_evening = df_final[df_final["대여시간"].isin([17, 18, 19])]

퇴근 시간대에는 따릉이 이용건수와 **지하철 하차인원** 사이의 상관관계가 높게 나타났습니다. 이는 퇴근 시간대에 사람들이 지하철에서 내린 뒤, 목적지까지 이동하기 위해 따릉이를 이용하는 패턴과 연결해서 해석할 수 있습니다.

## 5.3 로그 변환을 활용한 시각화

지하철 승하차 인원은 역별 편차가 크기 때문에 로그 변환을 적용했습니다.

```python
df_final["승차인원_로그"] = np.log1p(df_final["승차인원"])
df_final["하차인원_로그"] = np.log1p(df_final["하차인원"])
```

로그 변환 후 회귀선을 포함한 산점도를 그려 승하차 인원과 따릉이 이용건수의 관계를 시각적으로 확인했습니다.

```python
sns.regplot(
    data=df_final,
    x="하차인원_로그",
    y="이용건수",
    scatter_kws={"alpha": 0.5}
)
plt.title("지하철 하차인원(로그)과 따릉이 이용건수 관계")
plt.show()
```

로그 변환을 사용한 이유는 일부 대형 역의 승하차 인원이 지나치게 커서 그래프 해석이 어려워지는 문제를 줄이기 위해서입니다.

## 6. 추가 지표: 하차 인원 대비 따릉이 이용률

단순 따릉이 이용건수만 보면 유동인구가 많은 역이 항상 상위권에 위치할 가능성이 큽니다. 따라서 하차 인원 대비 따릉이 이용 비율을 계산하면, **지하철에서 내린 사람 중 따릉이 이용으로 연결될 가능성이 높은 역**을 비교할 수 있습니다.

```python
df_final["하차대비_따릉이이용비율"] = (
    df_final["이용건수"] / df_final["하차인원"]
)

df_final["하차대비_따릉이이용비율_%"] = (
    df_final["하차대비_따릉이이용비율"] * 100
)

df_final = df_final.replace([np.inf, -np.inf], np.nan).fillna(0)
```

### 지표 해석

| 지표 | 의미 |
|---|---|
| `이용건수` | 해당 역 주변의 따릉이 이용 규모 |
| `하차인원` | 해당 시간대 지하철에서 내린 인원 |
| `하차대비_따릉이이용비율_%` | 하차 인원 대비 따릉이 이용 비중 |

이 비율이 높은 역은 단순히 사람이 많은 역이라기보다, 지하철 하차 이후 따릉이로 이동하는 수요가 상대적으로 강한 역으로 해석할 수 있습니다.

## 7. 출퇴근 시간대와 비출퇴근 시간대 이용률 계산

출퇴근 시간대와 그 외 시간대를 나누어 시간당 따릉이 이용률을 계산했습니다.

```python
COMMUTE_HOURS = [7, 8, 9, 17, 18, 19]

# 시간대 구분
df_final["시간대_구분"] = np.where(
    df_final["대여시간"].isin(COMMUTE_HOURS),
    "출퇴근 시간대",
    "비출퇴근 시간대"
)

# 출퇴근 시간대 여부별 합산
usage_by_time_type = (
    df_final
    .groupby("시간대_구분", as_index=False)
    .agg(
        따릉이_총이용건수=("이용건수", "sum"),
        지하철_총하차인원=("하차인원", "sum"),
        관측시간수=("대여시간", "nunique")
    )
)

usage_by_time_type["시간당_따릉이이용건수"] = (
    usage_by_time_type["따릉이_총이용건수"] / usage_by_time_type["관측시간수"]
)

usage_by_time_type["하차대비_따릉이이용비율_%"] = (
    usage_by_time_type["따릉이_총이용건수"]
    / usage_by_time_type["지하철_총하차인원"]
    * 100
)

usage_by_time_type
```

출근 시간대, 퇴근 시간대, 그 외 시간대를 더 세분화하려면 다음과 같이 구분할 수 있습니다.

```python
def classify_time(hour):
    if hour in [7, 8, 9]:
        return "출근 시간대"
    elif hour in [17, 18, 19]:
        return "퇴근 시간대"
    else:
        return "비출퇴근 시간대"


df_final["시간대_구분"] = df_final["대여시간"].apply(classify_time)

time_type_summary = (
    df_final
    .groupby("시간대_구분", as_index=False)
    .agg(
        따릉이_총이용건수=("이용건수", "sum"),
        지하철_총승차인원=("승차인원", "sum"),
        지하철_총하차인원=("하차인원", "sum"),
        관측시간수=("대여시간", "nunique")
    )
)

time_type_summary["시간당_따릉이이용건수"] = (
    time_type_summary["따릉이_총이용건수"] / time_type_summary["관측시간수"]
)

time_type_summary["하차대비_따릉이이용비율_%"] = (
    time_type_summary["따릉이_총이용건수"]
    / time_type_summary["지하철_총하차인원"]
    * 100
)

time_type_summary = time_type_summary.replace([np.inf, -np.inf], np.nan).fillna(0)
time_type_summary
```

## 10. 분석 결과 요약

### 10.1 전체 시간대 기준

전체 시간대를 기준으로 보면 따릉이 이용건수와 지하철 승차인원, 하차인원은 모두 약한 양의 상관관계를 보였습니다.

| 비교 항목 | 상관계수 | 해석 |
|---|---:|---|
| 따릉이 이용건수 ↔ 승차인원 | 0.204090 | 약한 양의 상관관계 |
| 따릉이 이용건수 ↔ 하차인원 | 0.236029 | 약한 양의 상관관계 |

즉, 지하철 이용객이 많은 역일수록 따릉이 이용이 증가하는 경향은 있지만, 전체 데이터만으로는 강한 관계를 확인하기 어렵습니다.

### 10.2 출근 시간대 기준

출근 시간대에는 따릉이 이용건수와 지하철 승차인원 사이의 상관계수가 0.424731로 나타났습니다.

이는 출근 시간에 따릉이를 이용해 지하철역까지 이동한 뒤 지하철을 타는 패턴을 설명할 수 있습니다.

### 10.3 퇴근 시간대 기준

퇴근 시간대에는 따릉이 이용건수와 지하철 하차인원 사이의 상관계수가 0.625515로 나타났습니다.

이는 퇴근 시간에 지하철에서 내린 뒤 따릉이를 이용해 최종 목적지까지 이동하는 패턴을 설명할 수 있습니다.

## 11. 결론

이번 분석을 통해 다음과 같은 결론을 도출할 수 있습니다.

1. 지하철 이용객 수와 따릉이 이용건수는 전체적으로 약한 양의 상관관계를 보인다.
2. 출근 시간대에는 따릉이 이용건수가 지하철 승차인원과 더 관련이 있다.
3. 퇴근 시간대에는 따릉이 이용건수가 지하철 하차인원과 더 관련이 있다.
4. 따라서 따릉이는 출근 시간에는 지하철역으로 이동하는 수단, 퇴근 시간에는 지하철역에서 목적지까지 이동하는 수단으로 활용될 가능성이 있다.
5. 단순 이용건수보다 `하차대비_따릉이이용비율_%`를 함께 보면, 유동인구 대비 따릉이 연계 수요가 높은 역을 찾을 수 있다.
